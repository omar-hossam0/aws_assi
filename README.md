# AWS Assignment — Neural Network (tanh)

هذا المستودع يحتوي على ملف `aws.py` الذي ينفّذ شبكة عصبية بسيطة بطبقتين وتستخدم دالة التنشيط `tanh`.

المحتويات:
- `aws.py`: مثال لشبكة عصبية صغيرة، يحدد أوزانًا عشوائية من المجال [-0.5, 0.5]، والانحيازات `b1 = 0.5` و `b2 = 0.7`.

المتطلبات:
- Python 3.8+
- see `requirements.txt`

التشغيل:

```bash
pip install -r requirements.txt
python aws.py
```

ملاحظة: تم ضبط `numpy` كحزمة واحدة لازمة لتشغيل السكربت.
